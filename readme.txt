En este repositorio vendrán guías que explican detalladamente la administración de sistemas.

Métodos SSH con AWS ----> Diferentes formas de conexión y administración con ssh sobre máquinas remotas.

Consola CLI AWS ----> Explicación breve de uso del CLI de AWS y automatización mediante scripts de la creación de instancias EC2.

Docker AWS ----> Guía de uso de instalar y usar Docker en AWS.

Administración remota de usuarios y grupos ----> Guía para sincronizar usuarios y grupos entre Windows server y Ubuntu mediante Powershell.
